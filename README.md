# ProjectRecombination

### 프로젝트 소개

팀 단위 개발 경험을 쌓기 위해 진행하는 싱글 TPS 게임 개발 팀 프로젝트.

실제 프로젝트는 SVN을 사용하여 팀원과 공유하며, Github는 프로젝트 백업 용도로 사용하였습니다.
